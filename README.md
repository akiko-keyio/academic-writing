# academic-writing

A portable skill for refining academic writing for high-impact peer-reviewed STEM journals. Works with any AI assistant that supports skills.

## Usage

Hand the agent any section of a paper. The agent will:

1. **Establish key points** — extract the ideas the reader must take away, verify each is independent and necessary, and confirm they form a coherent argument when ordered. Halt if the foundation is unsound.
2. **Support key points** — flag every gap in evidence (missing citations, unsupported claims, logical leaps) for the author to address.
3. **Refine language** — rewrite for precision first, then efficiency. Verify the refined text still preserves every key point.

## Output

When Step 3 completes:

1. Refined text.
2. Change notes.
3. Flags for evidence gaps and remaining issues.

If Step 1 halts (the draft lacks clear or coherent key points), the agent returns only the issues the author must clarify or supplement.
